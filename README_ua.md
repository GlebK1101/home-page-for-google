# home-page-for-google
Персоналізована стартова сторінка для організації закладок. Дозволяє організовувати посилання та нотатки в таблиці за вкладками. Особливості: темна/світла теми, авто-отримання назв YouTube-відео, експорт/імпорт даних (JSON), редагування записів. Працює без сервера на LocalStorage — просто відкрийте файл у Chrome.

## Простий і зручний інструмент для організації ваших посилань та нотаток. Працює локально в браузері, без серверів та реєстрації.

### Проблема: «Вкладочне божевілля»
У всіх нас це було: ви відкриваєте браузер, і там висить 50 відкритих вкладок.
- "Цю статтю я прочитаю пізніше".
- "Це відео треба глянути ввечері".
- "А це посилання потрібне мені для роботи наступного тижня".

В результаті: браузер гальмує, оперативна пам'ять з'їдається, а знайти потрібне посилання в цій каші неможливо. Ви боїтеся закрити вкладку, щоб не втратити її назавжди. *(Так, я знаю про звичайні закладки, але мова не про них)*.

### Рішення
Цей проект — **один HTML-файл**, який замінює вам стандартну стартову сторінку.
Замість того, щоб тримати вкладку відкритою, ви просто копіюєте посилання сюди, додаєте нотатку і **зі спокійною душею закриваєте вкладку**. Всі ваші посилання акуратно розкладені по поличках (табах), а браузер працює швидко.

### Можливості
1. **Організація за вкладками**: Створюйте категорії (наприклад: "Робота", "YouTube", "Рецепти", "Почитати").
2. **Розумне додавання**: Вставте посилання на YouTube, і назва відео підтягнеться автоматично.
3. **Приватність**: Всі дані зберігаються тільки у вашому браузері (LocalStorage). Ніякої відправки даних на сторонні сервери.
4. **Кастомізація**: Темна і світла теми, налаштування розміру шрифту.
5. **Безпека даних**: Функція експорту та імпорту всіх даних у JSON файл (зручно для перенесення на інший ПК або створення бекапу).
6. **Ніякої установки**: Просто завантажте файл і відкрийте його в браузері.

### Як запустити
1. Завантажте файл `index_ua.html` з цього репозиторію.
2. Відкрийте його в Google Chrome (або будь-якому іншому браузері).
3. *(Рекомендується)* Зайдіть у налаштування браузера *(три крапки -> Зовнішній вигляд -> Показати кнопку «Домашня сторінка»)* і встановіть цей файл як **Домашню сторінку**.

### Важливо: Що вам потрібно знати
1. **Не чистіть «Дані сайтів» бездумно**: Якщо ви вирішите очистити історію браузера і поставите галочку навпроти *"Файли cookie та інші дані сайтів"*, ваші записи видаляться.
2. **Режим Інкогніто**: Якщо ви відкриєте файл у режимі Інкогніто, записи, зроблені там, зникнуть відразу після закриття вікна.
3. **Різні браузери = Різні дані**: Якщо ви відкриєте файл у Chrome, а потім у Firefox — ви побачите порожню сторінку у Firefox. Дані прив'язані до конкретного браузера.

### Як себе убезпечити
У проекті є вбудована функція резервного копіювання:
1. Натисніть на велику шестерню (Налаштування).
2. Натисніть "Експорт всього в JSON".
3. Збережіть файл до себе на комп'ютер.

*Аналогічно така ж функція є і для кожної вкладки окремо.*

Насолоджуйтесь порядком!
